Institutions and Bodies of the European Union

About

General

This dataset defines Linked Data compatible URIs for major institutions and bodies of the European Union that can be used as identifiers in EU-related datasets.

This dataset defines Linked Data compatible URIs for major institutions and bodies of the European Union. These URIs, identifying organisations on the European level, can be used as canonical identifiers in datasets.

Updates

The dataset is manually updated and has last been modified on 5th August 2012.

Formats and Examples

An example entity of this dataset is the European Commission with the URI: http://institutions.publicdata.eu/#ec.

Statistics

The dataset contains 643 triples with 65 distinct entities in total and has the following data links:

  • It links with owl:sameAs to DBpedia.
  • It links with owl:sameAs to EU Agencies and decentralized bodies (Agencies).

Vocabularies

In the dataset we use the following vocabularies:

  • Core Organization Ontology
  • Dublin Core
  • FOAF
  • SKOS
